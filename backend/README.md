## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## OpenAI integration (optional)

This project supports an optional OpenAI fallback for chat responses. The fallback will only be used when a local knowledge-base match is not confident enough and an API key is configured.

To enable, add the following to your `backend/.env` (or system environment):

```
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o-mini  # optional, defaults to gpt-4o-mini
```

Notes:
- OpenAI usage requires an API key and may incur charges depending on your account and model usage.
- If no API key is provided, the system will use the local FAQs (`resources/faqs.json`) only.
- The integration is designed to be conservative: the model is instructed to answer only from the KB and to reply with a canned fallback message if the KB doesn't contain an answer.

Hugging Face / Meta Llama support
---------------------------------

You can also use Meta's Llama models (for example `meta-llama/Llama-2-7b-chat-hf`) via the Hugging Face Inference API. To enable, add:

```
HUGGINGFACE_API_KEY=hf_...   # your HF token
HUGGINGFACE_MODEL=meta-llama/Llama-2-7b-chat-hf
LLM_PROVIDER=huggingface    # optional: prefer HF over OpenAI
```

Notes:
- Hugging Face hosted inference may require a subscription or quota depending on the model. Check Hugging Face pricing and model license before using.
- If you set `LLM_PROVIDER=huggingface` or configure `HUGGINGFACE_MODEL` to a Llama model, the chat controller will prefer HF. If HF fails and OpenAI is configured, it will fallback to OpenAI.
